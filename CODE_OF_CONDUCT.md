# Code de conduite

Ce projet adhere au [Contributor Covenant](https://www.contributor-covenant.org/fr/version/2/1/code_of_conduct/)
version 2.1.

En participant, vous vous engagez a maintenir un environnement ouvert, accueillant et
respectueux. Les comportements inacceptables (harcelement, propos discriminatoires, attaques
personnelles) peuvent etre signales au mainteneur, qui prendra les mesures appropriees.
