<h1 align="center">Empreinte</h1>

<p align="center">
  Assistant GenAI <strong>multimodal on-premise</strong> pour le reporting ESG/CSRD.
</p>

---

> 🚧 Initialisation du projet — le MVP arrive via la branche <code>develop</code>.

Empreinte transformera des documents RSE heterogenes (factures d'energie, certificats,
rapports PDF) en bilan carbone reglementaire, auditable et sourcé : extraction vision
on-premise, taxonomie ESRS, moteur carbone deterministe.
